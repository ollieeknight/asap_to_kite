# Test Data

This directory contains test FASTQ files for validating asap_to_kite functionality.

## Structure

- `data1/`: Test samples from first sequencing run
  - `test1_S1_L003_R[1-3]_001.fastq.gz`: Sample test1, lane 3
  - `test1_S1_L004_R[1-3]_001.fastq.gz`: Sample test1, lane 4
  - `test2_S1_L001_R[1-3]_001.fastq.gz`: Sample test2, lane 1
  - `test2_S1_L002_R[1-3]_001.fastq.gz`: Sample test2, lane 2

- `data2/`: Test samples from second sequencing run
  - `test2_S1_L003_R[1-3]_001.fastq.gz`: Sample test2, lane 3
